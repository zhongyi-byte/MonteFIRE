# 退休模拟器 (Retirement Simulator)

基于蒙特卡洛模拟的财务独立与提前退休 (FIRE) 计算器。

## 功能
- **蒙特卡洛模拟**：基于资产回报波动和通胀波动的随机模拟。
- **参数自定义**：支持年龄、资产、收支、被动收入等多种参数调整。
- **可视化图表**：展示破产概率和不同置信度下的资产走势。
- **对比分析**：自动对比推荐退休年龄及固定年龄（30/40岁）的走势。

## 运行环境
- Python 3.9+
- Flask, Numpy, Pandas, Matplotlib

## 本地启动
1. 安装依赖: `pip install -r requirements.txt`
2. 启动服务: `python app.py`
3. 访问: `http://127.0.0.1:5001`
